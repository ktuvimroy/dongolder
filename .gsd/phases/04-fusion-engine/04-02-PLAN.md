   ```python

   ```

2. Add class constants:
   ```python
   MIN_CONFIDENCE = 0.50  # 50% minimum to generate signal
   CONFLICT_PENALTY = 0.05  # 5% penalty per conflicting indicator
   ```

3. Update `__init__` to create FusionEngine:
   ```python
   self.fusion_engine = FusionEngine()
   ```

4. Replace `generate_signal()` logic:
   - Remove old MIN_SIGNAL_COUNT check
   - Call `self.fusion_engine.fuse(snapshot, current_price, support, resistance)`
   - Calculate confidence:
     ```python
     # Base confidence from fusion score
     if fusion.direction == SignalDirection.BULLISH:
         base_confidence = fusion.bullish_score
     elif fusion.direction == SignalDirection.BEARISH:
         base_confidence = fusion.bearish_score
     else:
         return None  # NEUTRAL = no signal
     
     # Apply conflict penalty
     conflict_penalty = len(fusion.conflicting_indicators) * self.CONFLICT_PENALTY
     confidence = max(0.0, min(1.0, base_confidence - conflict_penalty))
     
     # Filter by minimum confidence
     if confidence < self.MIN_CONFIDENCE:
         return None
     ```
   - Pass confidence to `_create_buy_signal` / `_create_sell_signal`

5. Update `_create_buy_signal()` and `_create_sell_signal()`:
   - Add `confidence: float` parameter
   - Pass it to RawSignal constructor:
     ```python
     return RawSignal(
         ...existing fields...,
         confidence=confidence,
     )
     ```

6. Update reasoning to include confidence context:
   - Add "Confidence: {confidence*100:.0f}% ({tier})" to reasoning list
   - Add "Indicators aligned: {fusion.aligned_indicators}" to reasoning
