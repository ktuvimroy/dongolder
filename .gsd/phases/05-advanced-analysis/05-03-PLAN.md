3. Add `fuse_with_advanced()` method to FusionEngine:

   ```python
   def fuse_with_advanced(
       self,
       snapshot: TechnicalSnapshot,
       current_price: float,
       support: PriceLevel | None = None,
       resistance: PriceLevel | None = None,
       sentiment: SentimentResult | None = None,
       ml_prediction: MLPrediction | None = None,
   ) -> FusionResult:
       """Fuse indicators with optional sentiment and ML inputs.
       
       This extends the base fuse() method to incorporate:
       - News sentiment analysis (10% weight when available)
       - ML pattern recognition predictions (10% weight when available)
       
       If sentiment/ML not provided, their weight is redistributed 
       proportionally to technical indicators.
       """
   ```

4. Implementation logic for fuse_with_advanced:
   - Start with base technical indicator scoring (like existing fuse())
   - If sentiment provided and has article_count > 0:
     - Add sentiment.score * self.weights.sentiment to bullish_score if BULLISH
     - Add to bearish_score if BEARISH
     - Track "Sentiment" in aligned/conflicting indicators
   - If ml_prediction provided and has confidence > 0.5:
     - Add ml_prediction.probability * self.weights.ml_pattern to appropriate score
     - Track "ML Pattern" in aligned/conflicting indicators
   - If sentiment/ML not available, scale technical weights to sum to 1.0
   - Apply S/R bonus (existing logic)
   - Return FusionResult with sentiment_contribution and ml_contribution fields
