  <files>
    src/gold_signal_bot/telegram/alerts.py
    src/gold_signal_bot/telegram/__init__.py
  </files>
  <action>
1. Create `src/gold_signal_bot/telegram/alerts.py`:
   - AlertManager class that orchestrates signal->telegram flow
   - Constructor takes:
     - TelegramBot instance
     - SignalGenerator instance
     - Optional: list of timeframes to check (default: [Timeframe.H1, Timeframe.H4])
   
   - `async check_and_alert() -> list[RawSignal]` method:
     - Iterate through configured timeframes
     - Call SignalGenerator.generate_signal(timeframe) for each
     - For each signal found, send via TelegramBot.send_signal()
     - Return list of signals that were sent
     - Log when signals are sent vs when no signal detected
   
   - `async run_continuous(interval_seconds: int = 900)` method:
     - Infinite loop that calls check_and_alert() every interval
     - Uses asyncio.sleep for async waiting
     - Catches and logs exceptions to prevent crash
     - This is the main entry point for running the bot
   
   - Track last signal per timeframe to avoid duplicate alerts:
     - Store dict[Timeframe, datetime] of last signal time
     - Don't re-alert same signal within 1 hour window
     - Compare timestamp and direction to detect duplicates

2. Update `src/gold_signal_bot/telegram/__init__.py`:
   - Add AlertManager to exports

Design notes:
- AlertManager is the "glue" between analysis and delivery
- It doesn't manage data fetching (that's the Scheduler's job)
- The expected flow is: Scheduler fetches data -> AlertManager checks for signals -> Sends to Telegram
- In MVP, user can run AlertManager.run_continuous() as main loop

   - Add `if __name__ == "__main__":` block to run main
   
   - Example structure:
     ```python
     async def main():
         """Run the gold signal bot."""
         settings = get_settings()
         
         # Setup data layer
         repo = OHLCRepository()
         
         # Setup analysis
         signal_gen = SignalGenerator(repo)
         
         # Setup telegram
         bot = TelegramBot(
             bot_token=settings.telegram_bot_token,
             chat_id=settings.telegram_chat_id
         )
         
         # Setup and run alerts
         alert_manager = AlertManager(bot, signal_gen)
         
         print("Gold Signal Bot starting...")
         print(f"Monitoring timeframes: H1, H4")
         print(f"Telegram chat: {settings.telegram_chat_id}")
         
         await alert_manager.run_continuous()
     
     if __name__ == "__main__":
         import asyncio
         asyncio.run(main())
     ```
