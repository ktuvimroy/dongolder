   - Export TelegramBot from bot.py

2. Create `src/gold_signal_bot/telegram/bot.py`:
   - TelegramBot class that wraps python-telegram-bot
   - Constructor takes bot_token and chat_id
   - `async send_message(text: str) -> bool` method
   - `async send_signal(signal: RawSignal) -> bool` convenience method
   - Uses HTML parse mode for formatting
   - Handle rate limiting: max 30 messages/sec (use simple async/await, no explicit rate limiting needed for MVP)

3. Create `src/gold_signal_bot/telegram/formatter.py`:
   - `format_signal(signal: RawSignal) -> str` function
   - Format includes:
     ```
     🟢 BUY SIGNAL - XAU/USD (or 🔴 SELL SIGNAL)
     ──────────────
     📈 Entry: $2,145.30
     🛑 Stop Loss: $2,134.60
     🎯 Take Profit 1: $2,166.70
     🎯 Take Profit 2: $2,188.00
     ──────────────
     📊 Timeframe: 4H
     ⏰ Generated: 2026-03-14 23:45 UTC
     
     💡 Reasoning:
     • RSI oversold at 28.5
     • MACD bullish crossover
     • Price near support at $2,140
     
     ⚠️ Risk/Reward: 1:2.0
     ```
   - Use HTML formatting tags (<b>, <i>) compatible with Telegram
   - Handle optional take_profit_2 gracefully
   - Format prices with 2 decimal places and $ symbol
   - Include all reasoning items as bullet points
  </action>
